# TennisFriendsBCN 🎾

Tennis rankings for your Barcelona friend group. Built with Next.js + Firebase.

## Tech Stack

- **Framework:** Next.js
- **Auth & DB:** Firebase (Auth + Firestore)
- **Rating System:** ELO

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy on [Vercel](https://vercel.com) — zero config for Next.js.
